AN-Alphabet National de Guinée (autres langues)

Version 1.0 Cross-platform Keyman keyboard for the National Alphabet of Guinea.

Overview

This keyboard provides input for the Alphabet National de Guinée (AN), a unified writing system serving multiple Guinean national languages including:

  • Poular (Pular Futa) — fuf
  • Kissi — kqs
  • Baga — bog
  • Guerzé (Kpelle) — gkp
  • Maninka — emk
  • Soussou (Susu) — sus

Installation

Windows

  1. Install Keyman for Windows.
  2. Double-click an_alphabet_national.kmp.
  3. Click Install.
  4. Switch to the AN keyboard via the language bar or Win + Space.

macOS

  1. Install Keyman for macOS.
  2. Open an_alphabet_national.kmp.
  3. Approve the input source in System Settings.
  4. Switch to the AN keyboard from the menu bar.

iOS

  1. Install Keyman for iOS from the App Store.
  2. Open the .kmp link or attachment.
  3. Tap Install.
  4. Enable the keyboard in Settings → General → Keyboard → Keyboards → Add New Keyboard.

Android

  1. Install Keyman for Android from Google Play.
  2. Open the .kmp attachment or link.
  3. Tap Install.
  4. Enable in Settings → System → Languages and Input.

Layout

The keyboard uses a modified AZERTY layout with the following 10/11/11/10 grid for mobile:

RowKeysContent
110AN native digits (1-9, 0)
211A · G · E · R · T · Y · U · I · O · P · £
311Q · S · D · F · Z · H · J · K · L · M · %
410W · X · C · V · B · N · ? · · · / · §
Note: G and Z are swapped from standard AZERTY positions. This is intentional for the AN layout.

Vowel Variants

Six vowel variants are accessible via AltGr (desktop) or long-press (mobile):

Base VowelVariant 1Variant 2
AAltGr+A → á—
EAltGr+E → éAltGr+Shift+E → ê
IAltGr+I → ï—
OAltGr+O → óAltGr+Shift+O → ô

Independent character: ù via AltGr+U.

Switching Keyboards

To switch between AN and your system's default keyboard (English/French), use the Globe key 🌐 on mobile, or your system shortcut on desktop:

  • Windows: Win + Space
  • macOS: Control + Space
  • iOS / Android: Globe (🌐) key on the keyboard

Font

The keyboard ships with the National Alphabet Regular font (NationalAlphabetRegular5.ttf). This font is required for the AN glyphs to display correctly. Without it, you will see standard Latin characters instead of AN characters.

The font is automatically installed by the Keyman package on all supported platforms.

Troubleshooting

I see Latin letters instead of AN characters

The National Alphabet font is not installed or active. Reinstall the keyboard package — the font should install automatically.

Backspace doesn't delete characters

This was a bug in earlier versions. Update to v1.0 or later.

Can't find the variant characters

On desktop, hold AltGr (Right Alt) and press the vowel key. On mobile, long-press the vowel key to see the variant menu.
